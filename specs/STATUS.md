## 🎯 MVP 목표

### 검증 가설

#### 생산 (콘텐츠 작성)
- **가설**: 경험담과 질문을 구분하면 글쓰기 목적이 명확해져 더 많은 글을 올릴 것
- **측정**: 전체 글 수, 경험담 수, 질문 수

#### 소비 (콘텐츠 읽기)
- **가설**: 나의 고민 주제에 맞는 글을 보여주면 더 많은 글을 읽을 것
- **측정**: 유저당 글 열람 수, 홈→상세 전환율

#### 만족도
- **가설**: 원하는 정보를 찾으면 저장할 것
- **측정**: 글 저장 수, 스크린샷 수, CSAT

### 성공 지표

| 카테고리 | 지표 | 목표 |
|----------|------|------|
| **생산** | 일일 신규 글 수 | 50+ |
| **생산** | 사용자당 평균 글 작성 수 | 2+ |
| **소비** | DAU | 100+ |
| **소비** | 사용자당 평균 글 열람 수 | 10+ |
| **소비** | 홈→글상세 전환율 | 30%+ |
| **소비** | 평균 체류 시간 | 5분+ |
| **만족도** | 일일 글 저장 수 | 200+ |
| **만족도** | 저장률 | 5%+ |
| **만족도** | 재방문율 | 30%+ |

## 📌 참고 사항

### 타겟 시장
무언가를 키우는 사람들 (육아, 반려동물)

### 핵심 가치
사람들의 상호작용을 기반으로 한 커머스 - 고민을 해결한 사람들(Giver)과 연결되어, 검증된 경험을 바탕으로 더 빠르고 쉽게 자신에게 맞는 제품을 발견하고 선택할 수 있는 탐색 중심의 소셜 커머스

### MVP 스코프 아웃
- ❌ 각 글에 '고민 제품' 직접 연결
- ❌ 커머스 기능 (상품 전시, 장바구니, 결제)
- ❌ 좋아요 기능
- ❌ 팔로우/팔로잉 기능
- ❌ 검색 기능
- ❌ 해시태그
- ❌ 글 공유하기
- ❌ 신고/차단 기능
